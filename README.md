# 🚀 Automated Network Request Management System - ServiceNow

An end-to-end **ServiceNow** application that automates the lifecycle of network service requests, from request submission to approval, task assignment, database updates, and user notifications. This project replaces manual request handling with an efficient, transparent, and automated workflow.

---

## 📖 Project Overview

Organizations often manage network-related requests through emails, spreadsheets, or manual ticketing systems, leading to delays, duplicate entries, and limited visibility.

This project leverages **ServiceNow** to provide a centralized platform where employees can submit network requests through a Service Catalog. Using **Flow Designer**, the system automates approvals, creates tasks for the network team, updates records, and sends real-time notifications, resulting in faster service delivery and improved operational efficiency.

---

## ✨ Features

- 📋 Service Catalog for network request submission
- ✅ Dynamic form validation using UI Policies
- 🔄 Automated approval workflow with Flow Designer
- 👨‍💼 Automatic task assignment to the Network Team
- 📧 Email notifications for request status updates
- 🗄️ Custom table to maintain approved network requests
- 📊 Request tracking and centralized record management

---

## 🛠️ Technologies Used

- **Platform:** ServiceNow Personal Developer Instance (PDI)
- **Workflow Automation:** Flow Designer
- **Modules:** Service Catalog, Service Portal
- **Scripting:** Client Scripts, UI Policies
- **Database:** Custom Tables
- **Notifications:** Email Notifications
